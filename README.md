## How to Use This Repository

1.  **Clone the Repository:** First, you need to clone the repository to get all the files.
2.  **Choose a Technology:** Browse the main directories to find a language or technology you want to practice.
3.  **Select a Project:** Inside the technology folder, choose a project that interests you.
4.  **Read the Instructions:** Each project folder contains its own `README.md` file. This file has all the information and steps you need to complete the project successfully.

### Important Note for Visual Studio Code Users

To properly view the instructions and explanations in the `README.md` files, it is recommended to use the preview feature.

In Visual Studio Code, right-click on the `README.md` file and select **"Open Preview"**. This will render the file in a readable format, showing images, links, and formatted text correctly.
