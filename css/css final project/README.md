### 1. Site Structure (4 Pages Minimum)

Your website must contain at least **four distinct HTML pages**. A navigation bar (menu) must be present on all pages to allow easy movement between them.

### 2. Design & Layout

- **Responsiveness:** The website must be fully responsive. It should look professional on desktops, tablets, and mobile phones.

- **Images:** Use images relevant to your topic.

### 3. Technology Stack

You may use any of the following methods to style your project:

- **Pure CSS** (using Flexbox and Grid).
- **Bootstrap.**
- **Tailwind CSS.**

> **Note:** Regardless of the framework you choose, your HTML code must be semantic and clean.
